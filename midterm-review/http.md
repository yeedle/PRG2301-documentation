# HTTP

## Topics

* HTTP methods (GET, PUT, PATCH, DELETE)
* headers (`Content-Type`, `Set-Cookie`, `CORS`)
* Status Codes (200, 400, 401, 404, etc)
* Cookies
* Inspecting requests with the network tab in devtools
